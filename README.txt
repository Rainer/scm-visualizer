scm-visualizer

PHP scripts to visualize source code management commits. You can write libraries to connect to any scm. Subversion and git will be provided.

You can although change the library for calculation and the library for creating the graph.

URL overrides will be implemented. Existing overrides:
calc_type= you can switch the calc_type with the url (e.g. commit or author)
